Multi-agent orchestration.
One transaction, every role.

GET /v1/transactions/{id}/events with Accept: text/event-stream. A payer agent fires pay, an auditor agent subscribes to the same transaction_id, a receipt-consumer agent waits for the receipt_ready event. Resumable via Last-Event-ID. Backed by Redis pub/sub, one channel per txn, TTL 1 h. Press play below — this is exactly what the wire looks like.

Auto-config Claude Desktop.
One JSON, six tools.

A user pastes https://flowlink.ink/.well-known/mcp.json into Claude Desktop's MCP settings. That manifest already exists — v0.2 wires the actual /mcp SSE endpoint behind it. Now Claude can create_invoice, pay_invoice, check_sanctions, get_receipt, etc. directly as tools — no copy-paste of curl commands, no out-of-band auth. SIWE happens once; JWT carried per session.

API keys for dev/test.
Scoped, revocable, in-app.

SIWE is the agent-default. But every llms.txt reader asks for a quick way to try without a wallet. v0.2 adds /dashboard/keys — the human flow that mints a scoped flk_test_* bearer. Backed by the ApiKey Prisma model that already exists. One reveal at mint time, then stored as bcrypt hash. Per-key scope picker, last-used timestamps, one-click revoke.
